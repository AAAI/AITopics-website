
Summary
-------

Extension module for Views RSS 2.x, providing additional set of iTunes podcast
feed elements, as defined in http://www.apple.com/itunes/podcasts/specs.html#rss


Installation
------------

1. Download/checkout and enable this module and Views RSS module in version 2.x
   (https://drupal.org/project/views_rss)
2. Follow the Views RSS and Views RSS: iTunes Elements module documentations
   for settings up a valid iTunes podcast feed:
   - Basic view setup: https://drupal.org/node/1346494
   - Setting up core <channel> elements: https://drupal.org/node/1344136
   - Setting up core <item> elements: https://drupal.org/node/1346498
   - Setting up iTunes podcast feed elements: https://drupal.org/node/1473728

Check out full Views RSS module documentation at https://drupal.org/node/1344136
